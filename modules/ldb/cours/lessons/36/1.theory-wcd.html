1. Création de la base de données et des tables

Énoncé :

Vous êtes chargé de créer une base de données pour gérer une ludothèque ou plus exactement une collection de jeux vidéo. :)

  1. Créez une base de données nommée collection_jeux_video.
  2. Créez les tables suivantes avec les contraintes appropriées :
    • plateformes : id_plateforme (clé primaire), nom_plateforme (chaîne, unique).
    • editeurs : id_editeur (clé primaire), nom_editeur (chaîne, non nul).
    • jeux : id_jeu (clé primaire), titre (chaîne, non nul), annee_sortie (entier), id_plateforme (clé étrangère vers plateformes), id_editeur (clé étrangère vers editeurs).
  3. Insérez au moins une plateforme, un éditeur et un jeu.

Exercice 2 : Requêtes SELECT avec jointures

Énoncé :

  • Ajoutez une nouvelle plateforme ("PlayStation 5"), un nouvel éditeur ("Square Enix") et un jeu ("Final Fantasy XVI", sorti en 2023).
  • Écrivez une requête pour afficher le titre du jeu, le nom de la plateforme et le nom de l’éditeur pour tous les jeux.

Exercice 3 : Fonctions de chaînes et de dates

Énoncé :

  • Créez une table joueurs avec id_joueur (clé primaire), pseudo (chaîne), date_inscription (date).
  • Insérez deux joueurs avec des pseudos et la date actuelle (01/04/2025 pour le premier et 08/03/2025 pour le second).
  • Écrivez une requête pour afficher les pseudos en majuscules et le nombre de jours depuis leur inscription.

Exercice 4 : Triggers

Énoncé :

  • Ajoutez une table inventaire pour suivre les jeux possédés par les joueurs : id_inventaire (clé primaire), id_joueur (clé étrangère), id_jeu (clé étrangère), date_achat (date).
  • Créez un trigger qui insère automatiquement la date actuelle lors de l’ajout d’un jeu dans l’inventaire si aucune date n’est spécifiée.
  • Testez le trigger.

Exercice 5 : Vues

Énoncé :

  • Créez une vue vue_collection_joueurs qui affiche le pseudo du joueur, le titre du jeu et la plateforme pour chaque entrée de l’inventaire.
  • Interrogez la vue après avoir ajouté un autre jeu dans l’inventaire d’un joueur.

Exercice 6 : Optimisation avec index

Énoncé :

  • Ajoutez un index sur la colonne titre de la table jeux.
  • Écrivez une requête pour rechercher tous les jeux dont le titre commence par "F" et expliquez pourquoi l’index peut aider.
