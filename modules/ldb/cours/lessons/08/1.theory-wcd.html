1. Introduction

Dans cette leçon nous allons explorer ensemble les interfaces du logiciel web phpMyAdmin (écrit en PHP) et de l'application DBeaver (écrite en Java). Ces deux outils sont des interfaces graphiques qui permettent de gérer des bases de données MySQL. Ils sont très utiles pour les développeurs web qui souhaitent manipuler des bases de données ou écrire des requêtes SQL dans un environnement intégré.

Vous trouverez une copie d'écran pour chaque application ci-dessous. Mais l'exploration complète des interfaces s'effectuera pendant le cours. :)
