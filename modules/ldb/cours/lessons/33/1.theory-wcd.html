1. Exercice : Création d'une base de données pour une boutique de vêtements avec MariaDB

Contexte

Vous êtes chargé(e) de concevoir une base de données pour une boutique en ligne qui vend des vêtements. Cette base doit permettre de gérer les produits (vêtements), les catégories de vêtements (par exemple, T-shirts, pantalons) et les commandes passées par les clients.

Instructions
  • Créez une base de données appelée BoutiqueVetements.
  • Créez trois tables avec les spécifications suivantes :
    • Une table Categories pour regrouper les types de vêtements.
    • Une table Produits pour lister les vêtements en vente.
    • Une table Commandes pour enregistrer les achats des clients.
  • Ajoutez au moins une contrainte (par exemple, clé primaire, clé étrangère, ou champ obligatoire).
  • Insérez quelques données d'exemple dans chaque table pour tester votre base.
Détails des tables

Table Categories :

  • id_categorie : Identifiant unique (entier, auto-incrémenté).
  • nom_categorie : Nom de la catégorie (chaîne de caractères, maximum 30 caractères, obligatoire).

Table Produits :

  • id_produit : Identifiant unique (entier, auto-incrémenté).
  • nom_produit : Nom du vêtement (chaîne de caractères, maximum 50 caractères, obligatoire).
  • prix : Prix du produit en euros (nombre décimal avec 2 chiffres après la virgule).
  • id_categorie : Identifiant de la catégorie du produit (clé étrangère liée à la table Categories).

Table Commandes :

  • id_commande : Identifiant unique (entier, auto-incrémenté).
  • id_produit : Identifiant du produit commandé (clé étrangère liée à la table Produits).
  • date_commande : Date de la commande (date).
  • quantite : Nombre d'unités commandées (entier, obligatoire).
Tâches à réaliser

1. Écrivez le code SQL pour :

  • Créer la base de données.
  • Créer les trois tables avec leurs colonnes et contraintes.
  • Insérer au moins 2 enregistrements dans chaque table.

2. Testez votre code dans un environnement MariaDB.

Données d'exemple à insérer

Dans Categories :

  • "T-shirts", "Pantalons".

Dans Produits :

  • "T-shirt blanc", 15.99 €, catégorie "T-shirts".
  • "Jean slim", 39.99 €, catégorie "Pantalons".

Dans Commandes :

  • Commande d’un "T-shirt blanc" (quantité : 2) datée du 01/04/2025.
  • Commande d’un "Jean slim" (quantité : 1) datée du 02/04/2025.

Ajouter une vue

Nous voulons créer une vue appelée Vue_Commandes_Detaillees qui affiche des informations combinées sur les commandes, les produits et leurs catégories. Cette vue permettra de voir facilement :

  • Le nom du produit commandé.
  • La catégorie du produit.
  • La quantité commandée.
  • La date de la commande.
