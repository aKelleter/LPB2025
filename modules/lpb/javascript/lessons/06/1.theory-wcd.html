En Javascript, les variables peuvent être classées selon leur déclaration et leur type de données. Lors de la leçon précédente, nous avons abordé leur déclacration, dans cette leçon, nous allons nous pencher sur les types de données que l'on peut leur attribuer.

1. Les types de données

En Javascript, il existe deux grandes catégories de types : les types primitifs et les types complexes.

  • Les types primitifs :
    Ces types représentent des valeurs immuables.
    1. String : Chaîne de caractères.
    2. Number : Nombres (entiers et flottants).
    3. BigInt : Les très grands nombres.
    4. Boolean : Les booléens représentent les valeurs logiques true et false.
    5. Undefined : Une variable déclarée mais non initialisée.
    6. Null : Représentant l'absence délibérée de valeur.
    7. Symbol : Représentant un identifiant unique.
  • Les types complexes :
    Ces types représentent des objets ou des collections mutables.
    1. Object : Structure de données clé-valeur.
    2. Array : Tableau de valeurs.
    3. Function : Bloque de code réutilisable.
    4. Date, Map, Set, ... : Objets spécialisés.

En résumé, JavaScript offre une grande flexibilité grâce à ses types dynamiques et ses diverses méthodes de déclaration, adaptées à différents besoins de programmation.
