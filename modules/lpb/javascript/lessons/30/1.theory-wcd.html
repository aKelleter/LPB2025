1. Introduction

En JavaScript, les tableaux (arrays) sont des outils puissants pour stocker et manipuler des données. Nous allons explorer des méthodes plus avancées pour modifier, rechercher, transformer et analyser les tableaux. Chaque méthode a un rôle spécifique, et les comprendre te permettra de programmer de manière plus efficace. Voici les méthodes que nous allons découvrir : splice, slice, concat, forEach, indexOf, lastIndexOf, includes, find, findIndex, findLastIndex, filter, map, sort, reverse, split, join, reduce, reduceRight, et isArray.

1. Méthodes pour modifier les tableaux

a) splice

Ajouter ou supprimer des éléments à une position précise
La méthode splice modifie un tableau en supprimant ou en ajoutant des éléments à partir d’un index donné.

  • Syntaxe : tableau.splice(indexDébut, nombreASupprimer, élémentsÀAjouter...)

b) slice

Extraire une partie du tableau
La méthode slice crée un nouveau tableau avec une portion du tableau original, sans le modifier.

  • Syntaxe : tableau.slice(indexDébut, indexFin)

c) concat

Fusionner des tableaux
La méthode concat combine deux tableaux ou plus en un nouveau tableau.

  • Syntaxe : tableau.concat(tableau2, tableau3, ...)

d) reverse

Inverser l’ordre des éléments
La méthode reverse inverse l’ordre des éléments dans le tableau (modifie le tableau original).

  • Syntaxe : tableau.reverse()

2. Méthodes pour parcourir et chercher dans les tableaux

e) forEach

Parcourir chaque élément
La méthode forEach exécute une fonction pour chaque élément du tableau.

  • Syntaxe : tableau.forEach((élément, index) => { ... })

f) indexOf

Trouver l’index d’un élément
La méthode indexOf renvoie le premier index où un élément est trouvé, ou -1 s’il n’existe pas.

  • Syntaxe : tableau.indexOf(élément)

g) lastIndexOf

Trouver la dernière occurrence
La méthode lastIndexOf renvoie le dernier index d’un élément.

  • Syntaxe : tableau.lastIndexOf(élément)

h) includes

Vérifier si un élément existe
La méthode includes renvoie true si l’élément est dans le tableau, sinon false.

  • Syntaxe : tableau.includes(élément)

i) find

Trouver un élément selon une condition
La méthode find renvoie le premier élément qui satisfait une condition.

  • Syntaxe : tableau.find((élément) => condition)

j) findIndex

Trouver l’index d’un élément selon une condition
La méthode findIndex renvoie l’index du premier élément qui satisfait une condition.

  • Syntaxe : tableau.findIndex((élément) => condition)

k) findLastIndex

Trouver le dernier index selon une condition
La méthode findLastIndex renvoie le dernier index qui satisfait une condition (nouveau depuis ES2022).

  • Syntaxe : tableau.findLastIndex((élément) => condition)

3. Méthodes pour filtrer et transformer les tableaux

l) filter

Filtrer les éléments
La méthode filter crée un nouveau tableau avec les éléments qui passent un test.

  • Syntaxe : tableau.filter((élément) => condition)

m) map

Transformer chaque élément
La méthode map crée un nouveau tableau en appliquant une transformation à chaque élément.

  • Syntaxe : tableau.map((élément) => transformation)

n) sort

Trier les éléments
La méthode sort trie les éléments du tableau (modifie le tableau original).

  • Syntaxe : tableau.sort((a, b) => comparaison)

Explications
Explication de la fonction de comparaison (a, b) => a - b, celle-ci est utilisée pour trier les nombres correctement :
  • Si a - b est négatif, a est placé avant b.
  • Si a - b est positif, b est placé avant a.
  • Si a - b est 0, l'ordre reste inchangé.
Exécution pas à pas du tri :
Avant le tri : [10, 5, 100].

Compare 10 et 5 → 10 - 5 = 5 → 5 est placé avant 10.
Compare 10 et 100 → 10 - 100 = -90 → 10 reste avant 100.

Résultat après le tri : [5, 10, 100].

4. Méthodes pour manipuler des chaînes et des tableaux

o) split

Convertir une chaîne en tableau
La méthode split divise une chaîne en un tableau selon un séparateur.

  • Syntaxe : chaîne.split(séparateur)

p) join

Convertir un tableau en chaîne
La méthode join combine les éléments d’un tableau en une chaîne avec un séparateur.

  • Syntaxe : tableau.join(séparateur)

5. Méthodes pour réduire les tableaux

q) reduce

Réduire à une seule valeur
La méthode reduce applique une fonction pour réduire le tableau à une seule valeur (de gauche à droite).

  • Syntaxe : tableau.reduce((accumulateur, élément) => opération, valeurInitiale)

r) reduceRight

Réduire de droite à gauche
La méthode reduceRight fonctionne comme reduce, mais de droite à gauche.

  • Syntaxe : tableau.reduceRight((accumulateur, élément) => opération, valeurInitiale)

6. Méthode pour vérifier un tableau

s) isArray

Vérifier si une variable est un tableau
La méthode Array.isArray renvoie true si la variable est un tableau, sinon false.

  • Syntaxe : Array.isArray(variable)

8. Conclusion

Ces méthodes te permettent de manipuler les tableaux de manière précise et puissante. Certaines modifient le tableau original (splice, sort, reverse), d’autres créent de nouveaux tableaux (slice, filter, map), et d’autres encore te donnent des informations (indexOf, includes). Pratique-les pour devenir à l’aise avec !
