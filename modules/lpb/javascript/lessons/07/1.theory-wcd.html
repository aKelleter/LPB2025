En JavaScript, le type string est utilisé pour représenter des chaînes de caractères. Une chaîne est une séquence de caractères entourée de guillemets simples ('), de guillemets doubles (") ou de backticks (`).

Création d'une chaîne

Voici trois façons principales de définir une chaîne :

Caractéristiques principales

  1. Immuabilité : Une fois créée, une chaîne ne peut pas être modifiée. Par exemple, vous ne pouvez pas changer un caractère directement dans une chaîne.
  2. Accès par index : Les caractères individuels d'une chaîne peuvent être accédés via des indices (zéro-indexé).
  3. Longueur : La propriété .length permet de connaître la longueur d'une chaîne.
  4. Concaténation : Vous pouvez concaténer des chaînes avec l'opérateur + ou +=.
  5. Template Literals : Avec les backticks (`), vous pouvez insérer des variables ou expressions directement dans une chaîne grâce à ${}.
