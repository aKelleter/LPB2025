La théorie sur les variables

En JavaScript, comme dans d'autres langages, une variable est un conteneur qui permet de stocker des données que vous pouvez réutiliser et manipuler dans votre code. Les variables permettent de donner un nom à ces données pour faciliter leur utilisation.

Pour déclarer une variable en JavaScript, nous allons devoir utiliser le mot clé let ou const (var n'est plus conseillé, mais nous allons expliquer la différence entre les deux dans la suite de cette leçon), suivi du nom qu’on souhaite donner à notre variable.

Règles à respecter :
• Le nom d’une variable doit obligatoirement commencer par une lettre ou un underscore (_) et ne doit pas commencer par un chiffre ;
• Le nom d’une variable ne doit contenir que des lettres, des chiffres et des underscores mais pas de caractères spéciaux ;
• Le nom d’une variable ne doit pas contenir d’espace.

De plus, notez que le nom des variables est sensible à la casse en JavaScript !
Enfin, sachez qu’il existe des noms « réservés » en JavaScript. Vous ne pouvez pas utiliser ces noms.

Vous pouvez également noter qu’on utilise généralement la convention lower camel case pour définir les noms de variable en Javascript. Cette convention stipule simplement que lorsqu’un nom de variable est composé de plusieurs mots, on colle les mots ensemble en utilisant une majuscule pour chaque mot sauf le premier. Par exemple, si je décide de nommer une variable « monage » j’écrirai en Javascript let monAge.

Déclaration d’une variable

Pour déclarer une variable en JavaScript, vous utilisez les mots-clés :

  • var (ancienne méthode, à éviter dans les nouveaux projets)
  • let (pour une portée limitée au bloc)
  • const (pour les valeurs constantes ou immuables)

Réassignabilité et redéclaration :
  • var : Vous pouvez réassigner et redéclarer une variable sans erreur, ce qui peut causer des bugs.
  • let : Vous pouvez réassigner, mais pas redéclarer dans la même portée.
  • const : Ni réassignable ni redéclarable.

  • var et let : Vous pouvez déclarer une variable sans l'initialiser.

3. Types de données pouvant être stockées

Une variable en JavaScript peut contenir différents types de données :
  • Primitifs :
    • String : Chaînes de caractères, ex. "Bonjour"
    • Number : Nombres, ex. 42, 3.14
    • Boolean : Valeurs booléennes, ex. true, false
    • Null : Une valeur spéciale représentant "rien"
    • Undefined : Une variable déclarée mais non initialisée
    • Symbol : Une valeur unique et immuable (rarement utilisé)
  • Objets :
    • Objets simples : { clé: valeur }
    • Tableaux : [1, 2, 3]
    • Fonctions : Une fonction est également un objet en JavaScript.

6. En résumé

  • Utilisez let pour des variables modifiables.
  • Utilisez const pour des constantes ou des références immuables.
  • Évitez var dans les projets modernes.
