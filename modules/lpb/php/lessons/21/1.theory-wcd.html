I. Présentation des tableaux associatifs

Un tableau associatif, c'est un tableau utilise des clés au format chaîne de caractères associées à une valeur. Ex : ["age" => 25].

Les tableaux associatifs vont s’avérer intéressant lorsque l’on souhaite donner du sens aux clés. Autrement dit, en créant une association forte entre les clés et les valeurs d’un tableau.
Exemple : 

        $personne = [
            "nom" => "Doe",
            "prenom" => "John",
            "age" => 30
        ];

II. Comment créer un tableau associatif

Les tableaux associatifs vont être différents des tableaux indexés, dans le sens où nous allons devoir définir chacune des clés, l'interprêteur PHP ne pouvant pas les nommer automatiquement. Comme pour les tableaux indexés, on peut créer un tableau en une fois en utilisant la structure de langage array() ou avec la syntaxe [] ou encore le construisant clé par clé et valeur par valeur.

III. Comment accéder à une valeur d'un tableau associatif

Pour accéder à une valeur d'un tableau associatif, on utilise la clé associée à cette valeur.
Exemple : 

        $personne = [
            "nom" => "Doe",
            "prenom" => "John",
            "age" => 30
        ];
        echo $personne["nom"]; // Affiche : Doe
Voir Exo2 du code source

IV. Parcourir les valeurs d'un tableau associatif

Pour parcourir les valeurs d'un tableau associatif, on peut utiliser une boucle foreach qui va parcourir chaque clé et valeur du tableau.
Exemple : 

        $personne = [
            "nom" => "Doe",
            "prenom" => "John",
            "age" => 30
        ];
        foreach($personne as $cle => $valeur) {
            echo $cle . " : " . $valeur . "
";
        }

V. Ajouter une valeur à un tableau associatif

Pour ajouter une valeur à un tableau associatif, on utilise la syntaxe suivante : 

        $personne = [
            "nom" => "Doe",
            "prenom" => "John",
            "age" => 30
        ];
        $personne["ville"] = "New York";
