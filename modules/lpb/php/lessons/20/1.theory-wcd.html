I. Introduction

Nous allons enfin aborder notre dernier type de variable, autrement dit les variables de type tableau ou array en anglais. Les tableaux vont être utile pour stocker plusieurs valeurs en même temps et vont donc se révéler très utiles pour stocker les données tabulaires (matrices).

II. Présentation des tableaux en PHP

En PHP, un tableau est une variable qui peut contenir plusieurs valeurs. Chaque valeur est associée à une clé, qui peut être un entier ou une chaîne de caractères. Les tableaux peuvent être de deux types :

  • Les tableaux indexés : chaque valeur est associée à un indice numérique.
  • Les tableaux associatifs : chaque valeur est associée à une clé.
En plus de cela les tableaux peuvent être multidimensionnels. Autrement dit, ils peuvent contenir d'autres tableaux.

III. Déclaration d'un tableau

Pour créer un tableau, on va soit utiliser :

  • la structure de langage array()
  • un syntaxe plus courte []

Lors de la déclaration on peut passer ou non un nombre variable d’arguments avec array() ou dans [].
Les arguments vont pouvoir être soit :

  • des valeurs simples (auquel cas les clefs seront des entiers générés automatiquement)
  • des paires clef => valeur

    
        $tableau = array(); // Tableau vide
        $tableau = []; // Tableau vide
        $tableau = array(1, 2, 3, 4, 5); // Tableau indexé
        $tableau = [1, 2, 3, 4, 5]; // Tableau indexé
        $tableau = array('cle1' => 1, 'cle2' => 2, 'cle3' => 3); // Tableau associatif
        $tableau = ['cle1' => 1, 'cle2' => 2, 'cle3' => 3]; // Tableau associatif

        $personnage = array('nom' => 'Link', 'type' => 'Epéiste', 'age' => 17); // Exemple avec un personnage

IV. Déclaration d’un tableau indexé (numérotés)

Les tableaux indexés sont les plus simple à créer. En effet, les clés vont être générées automatiquement par le PHP.

Pour créer un tableau indexé, on va utiliser la syntaxe suivante : 

        $tableau = [valeur1, valeur2, valeur3, ...];
Il suffit en fait d’indiquer une série de valeurs, séparées par des virgules et le une clé unique sera automatiquement associée à chaque valeur. La première clé aura pour valeur "0", la seconde clé la valeur "1", la troisième clé la valeur "2" et ainsi de suite. 
        // Indices :   0,       1,       2,    ...
        $tableau = [valeur1, valeur2, valeur3, ...];    
        Exemple : $tableau = [5, 10, 15];

Il est possible de créer des tableaux qui vont stocker des chaînes de caractères, des nombres, des booléens, etc. Comme d’habitude, seules les chaînes de caractères doivent être entourées de guillemets simples ou double.

Les clés automatiquement créées sont également appelées des indices ou index et sont associées aux différentes valeurs.

VI. Accéder aux valeurs d'un tableau indexé

Pour accéder à une valeur d’un tableau indexé, on va utiliser la syntaxe suivante : 

        $tableau[indice];
Exemple : 
        $tableau = [5, 10, 15];
        echo $tableau[0]; // Affiche 5
        echo $tableau[1]; // Affiche 10
        echo $tableau[2]; // Affiche 15

VII. Afficher un tableau à l'aide de boucles

Avec la boucle for

Nous allons boucler sur les indices de notre tableau et nous allons récupérer la valeur associée à l’indice en question à chaque nouveau passage dans la boucle.
Avant toute chose il nous faut une condition d'arrêt à notre boucle. Cette condition c'est "jusqu'à la fin du tableau". Il nous faut donc déterminer la taille de notre tableau, c’est à dire le nombre de valeurs contenues dans notre tableau.

La façon la plus simple de déterminer la taille d’un tableau est d’utiliser la fonction count() qui va prendre un tableau en argument et va retourner le nombre d’éléments de ce tableau.
Voici un exemple de boucle for qui va afficher les valeurs d’un tableau indexé : 

        $tableau = [5, 10, 15];
        for ($i = 0; $i < count($tableau); $i++) {
            echo $tableau[$i];
        }
