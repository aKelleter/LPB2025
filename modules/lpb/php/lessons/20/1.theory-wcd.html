I. Introduction

Nous allons enfin aborder notre dernier type de variable, autrement dit les variables de type tableau ou array en anglais. Les tableaux vont être utile pour stocker plusieurs valeurs en même temps et vont donc se révéler très utiles pour stocker les données tabulaires (matrices).

II. Présentation des tableaux en PHP

En PHP, un tableau est une variable qui peut contenir plusieurs valeurs. Chaque valeur est associée à une clé, qui peut être un entier ou une chaîne de caractères. Les tableaux peuvent être de deux types :

  • Les tableaux indexés : chaque valeur est associée à un indice numérique.
  • Les tableaux associatifs : chaque valeur est associée à une clé.
En plus de cela les tableaux peuvent être multidimensionnels. Autrement dit, ils peuvent contenir d'autres tableaux.

III. Déclaration d'un tableau

Pour créer un tableau, on va soit utiliser :

  • la structure de langage array()
  • un syntaxe plus courte []

Lors de la déclaration on peut passer ou non un nombre variable d’arguments avec array() ou dans [].
Les arguments vont pouvoir être soit :

  • des valeurs simples (auquel cas les clefs seront des entiers générés automatiquement)
  • des paires clef => valeur

    
        $tableau = array(); // Tableau vide
        $tableau = []; // Tableau vide
        $tableau = array(1, 2, 3, 4, 5); // Tableau indexé
        $tableau = [1, 2, 3, 4, 5]; // Tableau indexé
        $tableau = array('cle1' => 1, 'cle2' => 2, 'cle3' => 3); // Tableau associatif
        $tableau = ['cle1' => 1, 'cle2' => 2, 'cle3' => 3]; // Tableau associatif

        $personnage = array('nom' => 'Link', 'type' => 'Epéiste', 'age' => 17); // Exemple avec un personnage
