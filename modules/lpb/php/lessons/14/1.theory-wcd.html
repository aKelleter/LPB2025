I. La boucle for

La boucle for (« pour » en français) en PHP est plus complète, voir légèrement plus complexe à appréhender à priori que les boucles précédentes. Cependant, cette boucle est très puissante et sera majoritairement utilisée, avec la boucle foreach, dans nos scripts PHP. Vous devez donc la maîtriser :)

Le fonctionnement d’une boucle for se déroule selon trois phases :

  • Une phase d’initialisation ;
  • Une phase de test ;
  • Une phase d’incrémentation (ou décrémentation).
  • Dans la phase d’initialisation, on initialise la variable que nous allons utiliser dans la boucle. Elle est est exécutée, quoi qu'il arrive au début de la boucle.
  • Au début de chaque itération, la condition est évaluée. Si l'évaluation vaut true, la boucle continue et les commandes sont exécutées. Si l'évaluation vaut false, l'exécution de la boucle s'arrête
  • À la fin de chaque itération, la section "incrémentation" (ou décrémentation)est exécutée.

La syntaxe de la boucle for est la suivante : 

    for (initialisation ; condition ; incrémentation) {
        // Instructions
    }

A SAVOIR :
Les différentes sections entre les points virgules, peuvent éventuellement être laissées vides ou peuvent contenir plusieurs expressions séparées par des virgules. On peut intialiser 1 ou plusieurs variables dans la section initialisation et incrémenter 1 ou plusieurs variables dans la section incrémentation. Dans la section "condition" toutes les expressions séparées par une virgule sont évaluées mais le résultat est obtenu depuis la dernière partie.

II. Un petit mot sur la boucle foreach

La boucle PHP foreach est un peu particulière puisqu’elle a été créée pour fonctionner avec des variables de type tableaux (array), qui sont des variables spéciales que nous n’avons pas encore étudiées. Nous étudierons donc le fonctionnement de cette boucle en même temps que le type array (tableau).

III. Quel type de boucle utiliser?

Arrivé à ce stade, vous vous demandez peut-être quel type de boucle utiliser dans vos scripts PHP. Voici quelques conseils :

  • Si vous savez combien de fois vous devez répéter une action, utilisez une boucle for ;
  • Si vous avez une liste de valeurs à parcourir, utilisez une boucle foreach ;
  • Si vous ne savez pas combien de fois vous devez répéter une action, utilisez une boucle while ;
  • Si vous devez exécuter une action au moins une fois, utilisez une boucle do...while.
