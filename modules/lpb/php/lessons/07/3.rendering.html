Les opérateurs arithmétiques : + - * / % **
Addition: 2 + 4 = 6
Soustraction: 6 - 2 = 4
Multiplication: 5 * 3 = 15
Division: 15 / 3 = 5
Modulo (reste de la division): 5 % 2 = 1
Exposant: 2 ** 3 = 8

Les opérateurs de comparaison : == === != !== < > <= >=
$c = 5; $d = 3;

Les opérateur combinés arithmétiques et d'affectation : += -= *= /= %= **=
Valeur de $e au départ = 5
$e += 3; // $e = $e + 3; $e = 8
$e -= 2; // $e = $e - 2; $e = 6
$e *= 4; // $e = $e * 4; $e = 24
$e /= 2; // $e = $e / 2; $e = 12
$e %= 3; // $e = $e % 3; $e = 0
$e **= 2; // $e = $e ** 2; $e = 0
---------------
Les opérateurs de pré/post incrémentation et de pré/post décrémentation : ++ --
Valeur de $f au départ = 5
Pré-incrémentation: ++$f = 6
Valeur de $f après pré-incrémentation = 6
Post-incrémentation: $f++ = 6
Valeur de $f après post-incrémentation = 7
Pré-décrémentation: --$f = 6
Valeur de $f après pré-décrémentation = 6
Post-décrémentation: $f-- = 6
Valeur de $f après post-décrémentation = 5
---------------
Les opérateur logiques : and or xor && || !
$g = 5; $h = 3;
bool(true) $g == 5 and $h == 3
bool(true) $g == 5 or $h == 2
bool(true) $g == 5 xor $h == 2
bool(true) $g == 5 && $h == 3
bool(true) $g == 5 || $h == 2
bool(false) !($g == 5)
