Un opérateur, c'est est élément que l'on associe à une ou plusieurs valeurs pour produire une autre valeur (c'est une expression). Par exemple, l'opérateur + est associé à deux valeurs et produit une valeur qui est la somme des deux valeurs associées.

Les opérateurs peuvent être regroupés en fonction du nombre de valeurs qu'ils acceptent.
L'opérateur unaire n'opère que sur une seule valeur, par exemple ! (l'opérateur de négation) ou ++ (l'opérateur d'incrémentation).
Le second type, les opérateurs binaires (comme les très célèbres opérateurs mathématique + ou -) contient la plupart des opérateurs supportés par PHP.
Enfin, l'opérateur ternaire, ? :, qui accepte trois valeurs (on peut aussi l'appeler l'opérateur conditionnel).

Les opérateurs sont classés en plusieurs catégories:
  • Les opérateurs arithmétiques
  • Les opérateurs de comparaison
  • Les opérateurs logiques
  • Les opérateurs d'assignation / affectation
  • Les opérateurs de concaténation (opérateurs de chaînes de caractères)
  • Les opérateurs d'incrémentation et de décrémentation
  • Les opérateurs de type
  • Les opérateurs de pointeur
  • Les opérateurs de taille
  • Les opérateurs de bits

Les opérateurs de concaténation

. - .=
Il y a deux opérateurs de chaînes de caractères string. Le premier est l'opérateur de concaténation ., qui retourne la concaténation de ses deux arguments. Le second est l'opérateur d'affectation concaténant .=. Voir opérateurs d'affectation pour plus de détails.

Les opérateurs d'affectation

= += -=
L'opérateur d'affectation est utilisé pour assigner une valeur à une variable ou assigner une variable à la valeur d'une autre variable.
Une telle affectation de valeur est effectuée avec le signe =, ou caractère égal. Exemple: Les affectations peuvent également etre utilisées conjointement avec des opérateurs arithmétiques exemple + -.

On peut noter que l'affectation copie le contenu de la variable originale dans la nouvelle variable (affectation par valeur), ce qui fait que les changements de valeur d'une variable ne modifieront pas la valeur de l'autre. Cela peut se révéler important lors de la copie d'un grand tableau durant une boucle.

L'affectation par référence est aussi supportée, au moyen de la syntaxe: 

$var = &$othervar;
L'affectation par référence signifie que les deux variables pointent vers le même conteneur de donnée, rien n'est copié nulle part. le & fait référence à l'adresse mémoire de la variable.

Les opérateurs combinés arithmétique et affectation

+= -= *= /= %=

En programmation il est courant d'avoir à incrémenter une variable avec une certaine valeur. L'exemple le plus courant est la réalisation d'un compteur. Imaginons que vous Souhaitez incrémenter un compteur de 1 unité, vous devriez faire :

$compteur = $ compteur + 1;
Cependant, il y a une abréviation pour cela: 
$compteur += 1;
Cet opérateur combiné affectation/arithmétique accomplit la même tache. De façon générale, += et -= sont les opérateurs combinés le plus largement utilisés. Prochainement, nous verrons que pour cette opération il existe également un autre opérateur le ++ qui est l'opérateur d'incrémentation.

Signification
Exemple

-= Moins égal $x -= 4; $x = $x - 4;

Les opérateurs de pré / post incrémentation et décrémentation

Il existe une écriture encore plus courte pour la tache courante d'ajouter 1 ou de soustraire 1 d'une variable.
Pour ajouter 1 à une variable ou "incrémenter" utilisez l'opérateur ++: 

$x++; Qui équivaut à $x += 1; ou $x = $x + 1;
Pour soustraire 1 d'une variable, ou "décrémenter" utilisez l'opérateur --: 
$x--; Qui équivaut à $x -= 1; ou $x = $x - 1;
Avec cette écriture, vous pouvez spécifier si vous voulez que l'incrémentation ait lieu avant que la ligne de code soit exécutée ou après que la ligne ait été exécutée. Le code PHP ci-dessous va montrer la différence. 
    	   
        $x = 4;
        $x++; // post-increment 
        valeur de x est 4 
        
        $x = 4;
        ++$x; // pre-increment
        valeur de x est 5
