I. Présentation des tableaux multidimensionnels

Un tableau multidimensionnel est un tableau qui va lui-même contenir d’autres tableaux en valeurs.
Exemple : 

    $tableau = [
        [1, 2, 3],
        [4, 5, 6],
        [7, 8, 9]        
    ];

L'exemple ci-dessus est un tableau à deux dimensions. Tel qu'une feuille de calcul (ex. : excel), qui possède des lignes et des colonnes.
Les « sous » tableaux peuvent être des tableaux indexés ou des tableaux associatifs ou un mélange des deux. 

    $tableau = [
        [1, 2, 3],
        ['a' => 4, 'b' => 5, 'c' => 6],
        [7, 8, 9]        
    ];

Nous ne sommes pas limités dans le nombre de dimensions d’un tableau : on peut tout à fait déclarer des tableaux à 2, 3, 4, 5, n dimensions. Cependant, il est généralement déconseillé de créer trop de dimensions pour une question de lisibilté et de compréhension pour l'esprit humain.

NOTE : le nombre de dimensions d’un tableau indique le nombre d’indices nécessaires pour accéder à une valeur du tableau.

Exemple :
Pour accéder à la valeur 5 du tableau ci-dessus, on doit utiliser les indices 1 et 'b' : 

    echo $tableau[1]['b']; // affiche 5

II. Créer un tableau multidimensionnel

Pour créer un tableau multidimensionnel, on peut utiliser la syntaxe suivante : 

    $tableau = [
        [valeur1, valeur2, valeur3],
        [valeur4, valeur5, valeur6]      
    ];

    // ou

    $tableau = array(
        array("clé1"=>valeur1, "clé2"=>valeur2, "clé3"=>valeur3),
        array("clé4"=>valeur4, "clé5"=>valeur5, "clé6"=>valeur6)
    );

III. Parcourir un tableau multidimensionnel

Pour parcourir un tableau multidimensionnel, on peut utiliser 2 boucle foreach imbriquées.
Exemple : 

    $tableau = [
        [1, 2, 3],
        [4, 5, 6],
        [7, 8, 9]        
    ];

    foreach($tableau as $ligne) {
        foreach($ligne as $valeur) {
            echo $valeur . ' ';
        }
        echo '<br>'; // pour passer à la ligne suivante
    }

IV. Afficher rapidement le contenu de tableaux multidimensionnel ou non

Pour afficher rapidement le contenu d’un tableau, on peut utiliser la fonction print_r() ou var_dump().
Ces fonctions sont souvent utilisées pour le debugging.
Exemple : 

    $tableau = [
        [1, 2, 3],
        [4, 5, 6],
        [7, 8, 9]        
    ];

    print_r($tableau);
    echo '<br>';
    var_dump($tableau);

Personnellement, je crée souvent une fonction qui me pemet de choisir entre l'une ou l'autre comme ma fonction DBGPrint() : 

function DBGPrint(array $array, string $type = PR, string $info = null): void {

    if($type == PR)
    {
        echo '<pre>';
        echo (isset($info)?'<h4>' . $info . '</h4>':'');
        print_r($array);
        echo '</pre>';
    }elseif($type == VD)
    {
        echo '<pre>';
        echo (isset($info)?'<h4>' . $info . '</h4>':'');
        var_dump($array);
        echo '</pre>';
    }    
}
