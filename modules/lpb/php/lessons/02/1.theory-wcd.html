Comment utiliser des commentaires dans votre code source ?
Un commentaire dans un code PHP est une ligne qui n'est pas exécutée en tant que partie du programme. Son seul but est d'être lu par quelqu'un qui regarde le code.

Les commentaires peuvent être utilisés pour:
  • Permettre aux autres de comprendre votre code
  • Se rappeler ce que l'on a fait - La plupart des programmeurs ont déjà eu l'occasion de revenir sur leur propre travail un an ou deux plus tard et de devoir comprendre à nouveau ce qu'ils avaient fait. Les commentaires peuvent vous rappeler ce à quoi vous pensiez lorsque vous avez écrit le code
  • omettre certaines parties de votre code
Les commentaires peuvent être utilisés pour:
  • commenter une seule ligne
  • commenter sur plusieurs lignes

Les commentaires sur une seule ligne commencent par //.
Tout texte compris entre // et la fin de la ligne sera ignoré (ne sera pas exécuté).
Vous pouvez également utiliser # pour les commentaires d'une seule ligne, mais dans ce tutoriel, nous utiliserons //.

Les commentaires sur plusieurs lignes commencent par /* et finissent par */.
